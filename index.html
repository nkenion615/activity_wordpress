Exploring the WordPress Dashboard

Name: Najee Kenion

Date: 10/27/2025

Section Purpose
Posts Create and manage blog posts; organize with categories and tags.
Media Upload and manage images, video, and other media files.
Pages Create and manage static pages such as About or Contact.
Comments Review, approve, edit, or delete user comments left on posts.
Appearance Control the site theme, menus, widgets, and customize appearance.
Plugins Install, enable, disable, and configure plugins to add features.
Users Manage users, roles, and permissions for admins, editors, and subscribers.
Tools Utilities for importing/exporting content, site health, and maintenance tools.
Settings Global site options such as site title, permalinks, and reading settings.
